Lab Technician Help Documentation

Overview

Welcome to the Lab Technician Help Documentation. This guide will help you understand how to use the laboratory management features of the EHR system.

Dashboard

The Lab Technician Dashboard is your main workspace where you can:

  • View all lab orders that need processing
  • Search and filter lab orders by patient, test type, physician, date, and status
  • Enter results for pending lab orders
  • View details of specific lab orders

Processing Lab Orders

Viewing Lab Orders

On the dashboard, you'll see a table of lab orders. You can:

  • Click "View Details" to see more information about a specific order
  • Click "Enter Results" for orders that haven't been completed yet

Entering Lab Results

To enter results for a lab order:

  1. Click the "Enter Results" button next to the lab order
  2. Fill in the result data in the provided text area
  3. Click "Submit Results" to save and mark the order as completed

Viewing Lab Order Details

To view detailed information about a lab order:

  1. Click the "View Details" button next to the lab order
  2. You'll see patient information, test type, ordering physician, and any results if available

Search and Filter

You can search and filter lab orders using the search form on the dashboard:

  • Patient Name: Search by patient's first or last name
  • Test Type: Filter by specific test types (CBC, BMP, etc.)
  • Ordering Physician: Search by the physician who ordered the test
  • Date Range: Filter by order date range
  • Status: Filter by order status (Pending or Completed)

Click "Clear" to reset all filters and see all lab orders.

Profile Management

You can manage your profile information by clicking the "Profile" link in the navigation bar:

  • Update your first and last name
  • Change your password
  • View your assigned roles

Support

If you need additional help or encounter any issues:

  • Contact your system administrator
  • Refer to the general user manual
  • Submit a support ticket through the admin panel
